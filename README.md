# CRSF-X-1
A 3D printed plane focused on learning how Crossfire works, and coding my own program to recieve and transmit packets through an existing ELRS setup.

**Purpose** - 

The reason for this project is I recently got a ELRS transmitter/reciever kit, and figured I could save money by using an existing library to send CRSF packets, rather than buying a dedicated handset. Only after did I figure out that all the existing libraries were either just for recieving packets, or were severely outdated with bad documentation. That is why I figured I could make my own, and there's no better way to use it then a custom airframe and control surfaces!
